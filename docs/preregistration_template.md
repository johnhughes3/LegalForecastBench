## Rapid Run Minimum Fields

Rapid or provisional runs may omit final official-cycle details, but must include:

- release anchor;
- eligibility rule;
- exclusion rule;
- provisional-status label;
- manifest SHA-256;
- unit SHA-256;
- scorer SHA-256;
- model registry SHA-256.

## Notes and Amendments

Record any amendment before model evaluation starts. Amendments after labels or
model outputs exist should create a new cycle ID or a clearly labeled amended
run.
