# LegalForecast-MTD v0.1.0-alpha.1

Tag: `v0.1.0-alpha.1`
Python package version: `0.1.0a1`
License: Apache-2.0

This is a pre-data public alpha. It publishes the benchmark machinery and
methodology for review. It does not publish public cases, labels, model scores,
or an official leaderboard. Its result tier is `alpha-non-canonical`.

## Included

- Deterministic no-network fixture workflow:
  `uv run legalforecast fixture e2e --output-dir tmp/fixture-run`
- Typed artifact schemas for candidate discovery, retrieval, extraction,
  unitization, labels, packets, runs, scores, run cards, and reports.
- Micro-Brier scoring, calibration summaries, cost/tool accounting, and
  leaderboard rendering.
- Preregistration, model-card, run-card, and result-tier references.
- Guarded production acquisition commands with no-paid defaults.

## Not Included

- No public MTD corpus.
- No audited live labels.
- No real model scores.
- No canonical leaderboard.
- No hosted bring-your-own-key runner.

## Release Gate

Run:

```bash
uv run scripts/alpha_release_check.py
```

The gate runs formatting, lint, type checking, tests, fixture E2E, package
build, installed wheel checks, installed sdist checks, and artifact validation.

Do not infer model quality or court-outcome distributions from this alpha. The
fixture leaderboard is only a pipeline smoke artifact.
