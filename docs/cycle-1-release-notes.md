# Cycle 1 Release Notes

## Human Baseline

Cycle 1 does not include a human-baseline arm. No human reviewers were recruited, no human forecasts were collected, and no human-vs-model paired bootstrap comparison is part of the cycle-1 leaderboard.

Readers should treat cycle-1 results as model, fitted-baseline, and ablation evidence only. Any future human-baseline comparison must follow [human-baseline-protocol.md](human-baseline-protocol.md), score only the shared packet subset, and publish paired clustered bootstrap intervals before making model-vs-human claims.
